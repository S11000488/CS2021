# CSS
## 網頁美化
是cascading style sheets的縮寫
css涉及字型、顏色、邊距、高度、寬度、背景影象、高階定位等方面。它可以省去你大量時間
### <style>
  - p{
  text-align:center;
  color red: red;
  } 
  (置中,字體紅色)
